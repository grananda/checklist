# ui-mvp-checklist

UI del MVP (Fase 3, cierra F1): pantalla unica que consume la API de tareas-api-crud. Store Zustand (useTareasStore) con tareas y progreso derivado (calcularProgreso de shared); api/rest.ts. Componentes ListaTareas, ItemTarea (marcar/desmarcar, editar, borrar), FormularioTarea (alta y edicion con validacion, modal), IndicadorProgreso (role=progressbar), DialogoConfirmacion (borrado, foco atrapado) y EstadoVacio. Accesibilidad WCAG 2.1 AA (foco visible, focus-trap, aria-live, estado no solo por color), responsive/movil, textos en espanol. Tests de componentes (Vitest + Testing Library + axe) y e2e Playwright del recorrido MVP. Sin reordenar, reiniciar ni tiempo real. Basado en docs/detalle-historias-usuario.md HU-01..HU-06, docs/guia-estilos.md y docs/arquitectura-base.md 6.
