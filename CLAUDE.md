# CLAUDE.md — checklist (piloto AIDD)

Repositorio piloto para construir y validar el conjunto de skills **AIDD (AI Driven Development)** y la metodología AI-Native asociada.

## Metodología (referencia, solo lectura)

La metodología NO se carga automáticamente. Léela bajo demanda cuando trabajes el proceso de arquitecto IA o un skill AIDD:

- `.claude/methodology/native-ai-specs-sdd.md` — metodología v3 completa (referencia canónica).
- `.claude/methodology/native-ai-specs-sdd-getting-started.md` — guía de arranque rápido.

## Convención de carpetas

- `.claude/skills/` — skills del repo (AIDD + boosters). Es el "producto" de este piloto.
- `.claude/methodology/` — documentos que **explican** la metodología. Referencia estable, no se editan como parte de un proyecto.
- `docs/` — artefactos **generados por los skills** durante un proyecto (`cliente-requisitos.md`, `requisitos.md`, `mapa-historias-usuario.md`, ...). No mezclar con la metodología.
